Le langage HTML

HTML est un langage de balisage utilisé pour créer et structurer le contenu d'une page Web. Il est composé de balises (tags) qui décrivent la signification et la structure des éléments sur une page Web. Voici quelques concepts clés à retenir :

Balises HTML :

Les balises HTML sont des éléments entourés de chevrons ("<" et ">") qui indiquent au navigateur comment afficher le contenu. Par exemple, "h1" est une balise utilisée pour définir un titre de niveau 1, "p" est utilisé pour les paragraphes, et "a" est utilisé pour créer des liens.

Cours

Éléments :

Un élément HTML est constitué d'une balise d'ouverture, du contenu et d'une balise de fermeture.

Structure de base :

Une page HTML commence généralement par une balise <html> englobant tout le contenu, suivie d'une balise <head> contenant des informations sur la page (comme le titre), et d'une balise <body> contenant le contenu visible de la page.

Titres et paragraphes :

Les titres sont définis à l'aide des balises <h1> à <h6>, où <h1> est le titre le plus important et <h6> le moins important. Les paragraphes sont créés avec la balise <p>.

Les listes :

Vous pouvez créer des listes à puces avec la balise <ul> (unordered list) ou des listes numérotées avec la balise <ol> (ordered list). Les éléments de liste sont définis avec <li>.
